# Ten Heavens · 十重天

A bilingual map of the layered ontology of value: four worlds (Physical, Fictional, Real, On-chain), ten heavens layered through them, and the five canonical mappings (RWA, ETF, Cross-chain, ABS, Anchoring) that hold them together.

Built as a single deeply-designed page with:
- Interactive 4-world stack visualisation — click any heaven to read its essence.
- Per-world deep-dives with expandable layer panels.
- Mappings playground showing how value crosses between worlds.
- 5,000-year timeline from cowrie shells to ZK-native chains.
- Future-trends synthesis where the stack folds back onto mathematics.
- Fully bilingual EN ↔ 中文 with seamless toggle and `prefers-language` detection.
